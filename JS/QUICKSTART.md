# Quick Start

## Run

Open `index.html` in your browser (double-click or local server).

## API Key

Edit `index.html` and replace the API key:

```javascript
window.ewConfig = {
  apiKey: 'YOUR_API_KEY_HERE',
  customerId: 'SampleCustomer',
  appVersion: '1.0.0',
  debug: true
};
```

## Test

Click the buttons in the page to:
- Send logs at each severity level
- Measure timing of simulated operations
- Add breadcrumbs then trigger an error
- Set user identity
- Add tags
- Record metrics

View results in ExeWatch dashboard.
