# .NET Windows Forms Sample — Quick Start

1. Open `DotNetWindowsForms.csproj` in Visual Studio 2022 or later
2. Press F5 to run
3. Enter your API key (from https://exewatch.com) and Customer ID, then click "Use"
4. Explore the tabs: Logging, Timing, Device Info, Metrics, Updates
5. Check the ExeWatch dashboard to see logs, timings, and metrics
